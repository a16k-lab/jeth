# JETH product logos

These files are the canonical JETH product marks.

## Default usage

Use the orange icon for documentation, favicons, social previews, package and
extension listings, editor integrations, and other product-facing surfaces. Its
dark tile is designed to remain legible on both light and dark backgrounds.

Use `jeth-logo-monochrome.png` only when a neutral or print-oriented treatment
is explicitly required.

## Raster sizes

| File | Intended use |
| --- | --- |
| `jeth-orange-icon-32.png` | Favicons and compact UI glyphs |
| `jeth-orange-icon-128.png` | Documentation headers and editor file icons |
| `jeth-orange-icon-256.png` | README and medium product surfaces |
| `jeth-orange-icon-512.png` | High-density product surfaces |
| `jeth-orange-icon-1024.png` | Marketplace listings and social previews |

Use the smallest source that is at least as large as its rendered size. Keep the
artwork square and preserve its proportions. Do not recolor, redraw, stretch, or
place unrelated artwork inside the mark.
